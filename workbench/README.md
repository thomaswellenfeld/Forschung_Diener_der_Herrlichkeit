# Workbench - Arbeitsbereich für Forschungen

## Zweck
Dieses Verzeichnis ist der Hauptarbeitsbereich für alle forensisch-wissenschaftlichen Forschungen im Bahaitum-Kontext.

## Struktur

### active/
Aktuelle Forschungsprojekte und Untersuchungen:
- **persons/** - Personenbezogene Forschungen
- **structures/** - Strukturanalysen von Organisationen
- **events/** - Ereignisanalysen
- **patterns/** - Mustererkennungsstudien
- **investigations/** - Spezielle Untersuchungen

### data/
Datensammlungen und Quellenmaterial:
- **raw/** - Rohdaten aus primären Quellen
- **processed/** - Aufbereitete und bereinigte Daten
- **verified/** - Validierte und verifizierte Informationen
- **sources/** - Quellen- und Referenzmaterial

### results/
Ergebnisse der Untersuchungen:
- **reports/** - Detaillierte Analyseberichte
- **summaries/** - Executive Summaries
- **evidence/** - Evidenz und Beweismaterial
- **conclusions/** - Schlussfolgerungen

### archive/
Archivierte abgeschlossene Forschungen:
- **by_year/** - Chronologische Archivierung
- **by_topic/** - Thematische Archivierung
- **by_person/** - Personenbezogene Archivierung

## Sicherheitsprotokolle

Alle Forschungen in diesem Verzeichnis müssen:
1. Mit MCP Tools durchgeführt werden
2. KI-Abwehr-Maßnahmen beachten
3. Gegen ASSUMPTIONS.txt validiert werden
4. Vollständigen Audit-Trail führen

## Nutzungshinweise

- Jede Untersuchung erstellt eigenen Ordner in `active/`
- Ergebnisse werden standardisiert in `results/` abgelegt
- Abgeschlossene Arbeiten werden nach `archive/` verschoben
- Sensitive Daten erfordern besondere Zugriffskontrollen
