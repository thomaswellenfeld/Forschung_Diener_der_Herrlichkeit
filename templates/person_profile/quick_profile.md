# Template: Standard-Personenprofil

## Zweck
Standardisiertes Template für operative Personenprofile.

## Schnellanalyse-Checkliste

### Grunddaten
- [ ] Name und alternative Schreibweisen
- [ ] Geburts-/Todesdaten (falls bekannt)
- [ ] Geografische Verbindungen
- [ ] Wichtige Lebensstationen

### Bahaitum-Kontext
- [ ] Rolle im Bahaitum
- [ ] Zeitliche Einordnung
- [ ] Verbindungen zu Schlüsselfiguren
- [ ] Konstruktionsmerkmale

### Sicherheitsrelevanz
- [ ] Aktuelle Einflussnahme
- [ ] Netzwerkverbindungen
- [ ] Finanzströme
- [ ] Geheimdienstbeziehung

## Operative Auswertung

### Quick-Assessment
- [ ] Primärquellen identifiziert
- [ ] Widersprüche gefunden
- [ ] Muster erkennbar
- [ ] Bedrohungspotential

### Empfehlung
- [ ] Weiterführende Untersuchung erforderlich
- [ ] Beobachtung empfohlen
- [ ] Keine weitere Aktion nötig
